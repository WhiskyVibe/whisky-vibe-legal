# Whisky Vibe legal pages for GitHub Pages

Files:
- `index.html`
- `terms.html`
- `privacy.html`

## Recommended repo layout
Upload these files to the root of the GitHub Pages repository.

Example public URLs:
- `https://<org-or-user>.github.io/<repo>/terms.html`
- `https://<org-or-user>.github.io/<repo>/privacy.html`

## Important
- Replace `대표님 이메일 입력 필요` with the actual contact email before publishing.
- When TikTok asks for URL verification, upload the verification file they provide to the same repo root.
- If TikTok asks to verify a URL prefix, use the GitHub Pages URL prefix, not a GitHub wiki URL.
